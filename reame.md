# But du projet 
## principale 
Il est bien plus saisissant de prendre conscience du risque de ne rien gagner, même après 1000 ans de jeu, trois fois par semaine. À mon sens, cette réalité est bien plus frappante que de simplement affirmer que nos chances de gagner sont de 1 sur 1 906 884. 
Trop souvent, nous sommes influencés par la présence de gagnants, sans réellement mesurer la chance extraordinaire qu'ils ont de remporter le gros lot de leur vivant.

- **Objectif** : Simuler les tirages du loto sur une période donnée en années.
- **Fréquence des tirages** : Le loto est présenté en 3 tirages par semaine.
- **Prix du ticket** : Le prix du ticket est de 2,20 euros.
- **Mécanique du jeu** : Le joueur doit choisir 5 chiffres sur 49 et une étoile sur 10.


# installation 
python version : 3.12.2
## MAC
```bash
virtualenv .env
source .env/bin/activate
pip install -r requirement.txt
```

## windows
```bash

.env/script/Activate
pip install -r requirement.txt
```
